
# Sensitive Content Design in Flutter

Este proyecto de Flutter presenta un diseño de interfaz de usuario que maneja contenido sensible y ofrece opciones para ocultar o mostrar dicho contenido. La aplicación utiliza Flutter para crear una interfaz atractiva y funcional.

## Características

- **Filtro de Contenido Sensible:** La aplicación utiliza un filtro de desenfoque para ocultar o mostrar contenido sensible según la preferencia del usuario.
- **Botón de Alternancia:** Un botón de alternancia permite a los usuarios cambiar entre ver y ocultar el contenido sensible.
- **Diseño Atractivo:** La interfaz de usuario se ha diseñado cuidadosamente para proporcionar una experiencia agradable al usuario.
